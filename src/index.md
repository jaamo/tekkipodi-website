# Welcome to Tekkipodi Website

This is a static website generated with [Eleventy](https://www.11ty.dev/) and markdown files.

## Features

- **Markdown Support**: Write your content in markdown
- **Static Generation**: Fast, secure, and easy to deploy
- **Flexible**: Use Nunjucks templates for layouts

## Getting Started

1. Install dependencies: `npm install`
2. Start development server: `npm run serve`
3. Build for production: `npm run build`

Check out the [About](/about/) page or the [Blog](/blog/) section to see more examples.
